# 12. Zero-Downtime Shutdown Analysis

The gRPC server has a `shutdown-grace: 120` second window.
The shutdown flow:
1. gRPC server stops accepting new connections (Phase MAX_INT)
2. In-flight gRPC calls complete (up to 120s)
3. `ResumeDirectPool.shutdownPool()` — waits `awaitTerminationTimeout` seconds for queued saga resumes
4. `SubmitDirectPool.shutdownPool()` — same
